# Employee SQL Project

This project is a simple SQL practice database created to understand and apply basic to intermediate SQL concepts using an Employee table.

---

## 📌 Project Overview

The database contains an `employee` table with sample employee records from different departments such as IT, HR, Finance, and Marketing.

It demonstrates how to perform common SQL operations like data insertion, retrieval, filtering, sorting, grouping, and aggregation.

---

## 🧱 Database Structure

### Table: employee

| Column      | Data Type    | Description               |
|------------|--------------|---------------------------|
| EmpId       | INT (PK)     | Unique employee ID        |
| FirstName   | VARCHAR(50)  | Employee first name       |
| LastName    | VARCHAR(50)  | Employee last name        |
| Department  | VARCHAR(50)  | Department name           |
| Salary      | INT          | Employee salary           |
| HireDate    | DATE         | Date of joining           |
